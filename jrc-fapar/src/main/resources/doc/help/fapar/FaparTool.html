FAPAR Processor

The FAPAR Processor provides a quantitative estimate of the Fraction of Absorbed Photosynthetically Active Radiation calculated from a given MERIS Level 1b product. The algorithm corresponds to the current MERIS Global Vegetation Index one which provides the land product in the actual Level 2 MERIS products.

This physically-based algorithm provides a direct measure of solar radiation absorbed and used in the photosynthetic process and can be used also as an indicator of the presence of vegetation. It can be ingested, in conjunction with other input, to estimate plant productivity or the exchange of carbon between the biosphere and the atmosphere.

A number of flags document information associated to the product.
